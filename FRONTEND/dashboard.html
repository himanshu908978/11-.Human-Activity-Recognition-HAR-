About this project

This is my Eleventh project in the domain of AI/ML/DL whose AI model is made by using the Pytorch and concepts of CNN(Convulational Neural Network) and trained on HMDB51 dataset.

This Activity Recogniser project detects whether there is an activity of labels in your given video.

This trained AI model detect the labels in image and gives you a label that it could find in your given image and also give a Confidence score that represents that how much confident model really is to give you that output.

Labels :-

  • walk
  • talk
  • stand
  • sit
  • eat
  • drink
  • laugh
  • smile

So if you upload a video that belongs to any of these label's category then you will get a predicted class from model in this scenario normal and pneumonia label and confidence score that shows the surety of predicted class.

Conclusion

I trained this model on google colab notebook's free GPU T4(it comes with a specfic duration of usage) for 10 epochs and then the minimum loss was about 1.98 something and i got 29% accuracy on testing dataset of HMDB51 dataset.

Test dataset metrics :-

  • 1).Accuracy :- 0.298001998001998
  • 2).F1 :- 0.2880000799168864
  • 3).Recall :- 0.2680657640232109

So final conclusion is that this model can predict the activity of video but of specific classes on which this model was trained.

Well if we looks at the HMDB51 dataset then we will find that there are 51 different classes in the dataset but i trained this model for only 8 classes and there is a reason because on google colab it took 2.5hour to train this model with only 8 classes and trained on only 20 frames.

And if we calculate or even compare this with 51 classes and with at least 60frames then it could take some 7-10days to train this model with approx accuracy on dataset of 50%-55% but for my condition this is impossible and even after this model will be still unreliable.

So i will consider this project as a learning project and i would not like to recommend this project's prediction as reliable predictions.
